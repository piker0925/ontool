# Security Policy

OnTool은 개인 프로젝트지만, 소셜 로그인·JWT 인증·관리자 페이지와 RSA/AES/Bcrypt 같은 암호화 도구를 실제로 다루고 있고 [배포된 서비스](https://dev-toolbox-ivory.vercel.app)로 운영 중입니다.

## 지원 범위

`main` 브랜치 기준 최신 배포 버전만 지원합니다. 별도로 관리되는 이전 버전은 없습니다.

## 취약점 신고

보안 취약점을 발견하셨다면 **공개 GitHub Issue로 올리지 말고** 아래 방법 중 하나로 알려주세요.

- 이메일: piker0925@gmail.com
- 또는 이 저장소의 [Private vulnerability reporting](../../security/advisories/new) 기능 사용

신고 시 아래 내용을 포함해주시면 대응이 빨라집니다:
- 재현 방법(가능하면 단계별로)
- 영향 범위(어떤 기능·데이터가 노출/조작 가능한지)
- 발견한 환경(브라우저, 계정 상태 등)

## 대응

1인 개발 프로젝트라 공식 SLA는 없지만, 신고를 받으면 최대한 빠르게 확인하고 조치합니다. 심각한 취약점은 확인 후 별도 공지 없이 조용히 패치할 수 있습니다 — 악용 가능한 정보가 먼저 공개되지 않도록 하기 위함입니다.
