# 2026년 요율/세액표 출처 기록

`/life` 생활 계산기(급여·금융·날짜)는 "공식 자료 기반, 근사식 금지"가 확정 방침(`.scratch/v2-feature-list.md` §3)이라, 코드에 들어가는 모든 수치의 출처를 여기 기록한다. **매년 갱신이 필요한 값들**이라 다음 연도 작업 시 이 문서부터 다시 확인할 것.

값 자체는 `front/src/utils/salaryRates2026.ts`, `front/src/utils/withholdingTaxTable2026.ts`에 있다 — 이 문서는 "그 값을 어디서·언제·어떻게 확인했는지"만 다룬다.

## 신뢰도 표시 기준

- 🟢 **확정**: 정부 기관 1차 출처(보도자료·고시 원문·법령)를 직접 확인함
- 🟡 **약함**: 2차 출처(집계 사이트·블로그)로만 확인, 1차 고시 원문을 못 찾음 — 여러 독립 출처가 일치하면 실사용은 가능하나 원문 확인이 남아있음

### 자동 수집 실패 기록 (참고)

이 표는 처음엔 자동으로 못 가져왔다 — 국세청/홈택스가 배포하는 간이세액표 PDF는 HWP 변환 특성상 텍스트 레이어가 깨져 있어(바이너리로만 읽힘) 자동 파싱이 안 됐고, law.go.kr 별표 페이지는 JS 렌더링이라 일반 fetch로는 빈 페이지만 나왔다. 최종적으로 사용자가 홈택스에서 엑셀 파일을 직접 받아 `docs/`에 넣어줘서 해결— 엑셀 내부는 XML 텍스트라 파싱이 가능했다.
